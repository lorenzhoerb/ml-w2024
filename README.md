# ML 2024

## Getting Started

1. Create a python virtual environment

```sh
python3 -m venv .venv
```

2. Activate virtual environment 

- Mac / Linux

```sh
source .venv/bin/activate
```

- Windows

```sh
.venv\Scripts\activate
```

3. Install requirements

```sh
pip install -r requirements.txt
```

4. Lauch Jupyter Lab

```sh
python -m jupyter lab
```
